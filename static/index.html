Graph + LaTeX = GraTeX

GraTeX

GraTeX allows you to turn the equations you enter into DesmosCaluculator into graphs and LaTeX images that you can save for use in other documents.
Type the formula into DesmosCaluculator and click "Convert".
The source code is available on GitHub.
